# 🐍 Retro Nokia Snake

A nostalgic recreation of the classic Nokia Snake game. This project focuses on **Canvas API** rendering, **Game Loop** management, and a custom **CRT filter** for that authentic 90s handheld feel.

-----

-----

## ✨ Features

  * **Retro Aesthetics:** Custom CSS "Nokio" phone case with CRT scanline effects and a pixelated LCD display.
  * **Authentic Logic:** Grid-based movement, self-collision detection, and random food generation.
  * **Keyboard & Touch Support:** Play using your keyboard arrows or the interactive on-screen D-Pad.
  * **High Score System:** Uses `localStorage` to save your best score even after closing the browser.
  * **Dynamic Audio:** Simple Beep-tone synthesis using the **Web Audio API**.

-----

## 🛠️ Tech Stack

  * **Graphics:** HTML5 Canvas API
  * **Styling:** Tailwind CSS & Custom CSS Keyframes
  * **Programming:** Vanilla JavaScript (RequestAnimationFrame Game Loop)
  * **Fonts:** VT323 (Pixel Font)

-----

## ⌨️ Controls

| Input | Action |
| :--- | :--- |
| **Arrow Keys** | Change Direction |
| **Enter / Start Button** | Start / Restart Game |
| **On-Screen D-Pad** | Fully functional for mobile/mouse users |
